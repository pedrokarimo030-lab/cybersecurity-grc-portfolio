# cybersecurity-grc-portfolio
Cybersecurity GRC portfolio demonstrating risk assessment, compliance, audit testing, and policy development aligned with NIST CSF, NIST 800-53, and ISO 27001.

## 🎯 Project Overview
This project simulates a real-world GRC program, focusing on identifying, assessing, and mitigating cybersecurity risks while aligning with established frameworks.

### Key Objectives:
- Identify and document cybersecurity risks
- Assess risks using a structured 5x5 risk matrix
- Develop security policies aligned with compliance frameworks
- Perform basic audit testing procedures
- Evaluate third-party/vendor risks

---

## 🧩 Project Components

### 1. Risk Register
- Identified key organizational risks
- Defined threats, vulnerabilities, likelihood, and impact
- Assigned risk owners and mitigation strategies

### 2. Risk Matrix (5x5)
- Applied Likelihood × Impact scoring model
- Categorized risks into Low, Moderate, High, and Critical
- Established clear risk response actions

### 3. Access Control Policy
- Implemented the Least Privilege principle
- Enforced Multi-Factor Authentication (MFA)
- Defined user access review processes
- Included account lifecycle management

### 4. Audit Testing
- Validated the effectiveness of access controls
- Simulated audit procedures
- Documented control testing results
